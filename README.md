# User Directory App

A simple Android application built using Jetpack Compose that displays a list of users fetched from a remote API and stored locally using Room Database. The app follows an offline-first architecture, meaning data persists even without an internet connection.

## App Overview

- Displays a list of users with basic details.
- Fetches user data from a public API.
- Saves and retrieves user data locally using Room.
- Clean and modern UI built with Jetpack Compose.
- Supports searching/filtering through the user list.

## Concepts Used

### Jetpack Compose
- Declarative UI framework
- Material 3 theming 

### Architecture/Data Layer
- Room Database for local storage
- Data Access Objects for database queries
- Repository pattern for managing data flow
- ViewModel for UI state and logic management
